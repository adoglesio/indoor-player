# Indoor Player – Reprodutor de Mídia Digital

Aplicação React que exibe playlists em TVs (modo quiosque/tela cheia) com suporte a pareamento via código de 6 dígitos.

## 🚀 Tecnologias
- React + TypeScript
- Supabase (banco de dados e autenticação)
- Capacitor (para gerar APK Android)

## 📋 Pré-requisitos
- Node.js 18+
- npm ou yarn

## 🔧 Instalação e Execução
```bash
git clone https://github.com/seu-usuario/indoor-player.git
cd indoor-player
npm install
# configure .env (idêntico ao CMS)
npm start
